1. Заполнить свои токены, id приложения и ключи доступа в файле "settings.ini"
2. Сгенерировать токен ВК поможет файл testAccess:
    После заполнения Ключа приложения, id приложения - выполнить код,
    сгенерируется ссылка на получение ключа авторизации. Вставить полученный ключ
    также в settings.ini, затем снова выполнить код, и пройти по второй ссылке,
    оттуда скопировать токен в файл settings.ini в соответствующую строку.
